# Diptyque Fragrance Selector 🌿

Интерактивная Unity-игра, вдохновлённая миром нишевой парфюмерии Diptyque.  
Проект сделан как небольшой digital-опыт, где пользователь выбирает аромат, собирая его из ингредиентов.

---

## О проекте

Игрок выбирает 3 ингредиента — например розу, дерево, цитрусы или дым — и на основе этой комбинации система формирует персональный аромат.

Каждая комбинация создаёт ощущение небольшого ритуала сборки аромата:
- выбор ингредиентов
- визуальная композиция
- появление результата
- раскрытие названия и нот

---

## Идея

Главная цель проекта — передать ощущение бренда Diptyque через интерактив:

- минимализм
- внимание к деталям
- ощущение редкости и персонализации
- мягкая, почти редакционная подача UI

Результат воспринимается не как игра, а как небольшой брендовый опыт.

---

## Как это работает

Пользователь:
- кликает на ингредиенты
- собирает комбинацию из 3 элементов
- получает аромат, который раскрывается как композиция

После этого появляется:
- название аромата
- описание
- ноты
- промокод как финальный жест
- визуальный центр композиции

---

## Визуальный стиль

Проект построен вокруг идеи овальной композиции, где текст и элементы словно обтекают центр.

Это создаёт ощущение:
- этикетки парфюма
- движения по кругу
- органичной композиции, а не интерфейса
